# Wireshark-Capture-Reads
This is a collection of analyzed Wireshark files. It includes screenshots from the files and findings.

## Projects 
- DVWA Command Injection: Analyzed a network capture of an OS command injection attack. Identified malicious payload and reconstructed credential exfiltration via Wireshark HTTP stream analysis.
- DVWA Brute Force: Analyzed a network capture of a credential brute force attack. Identified the attack pattern across 180 login attempts and determined the successful credentials through HTTP response size analysis.
